== Description ==

Cozy Addons is the powerful elementor widgets plugin, which comes with advance powerful widget with multiple elementor layout that powerup your website. Cozy Addons provides tons of advanced layouts for team, portfolio, blog, magazine, newsportal, latest news, popular news, sliders, course list, property list, travel paackage list, testimonials and many more. It comes with header and footer builder for elementor(pro version) and stikcy option for any elementor section, With it's advanced widgets, features and variety of layouts, you can create any kinds of stunning website for any niches whether you are personal brands, freelance or agency professional.

== Installation ==

= DIRECT FROM THE DASHBOARD =
1. Go to Dashboard ->Plugins -> Add New
2. Search for "Cozy Addons"
3.Now Install and Activate the Plugin

= DOWNLOAD & UPLOAD MANUALLY =
1. Go to https://wordpress.org/plugins/ and Search for "Cozy Addons" and download file.
2. Extract File and placed it to the “/wp-content/plugins/” directory.
3. Activate the plugin through the “Plugins” menu in WordPress.
